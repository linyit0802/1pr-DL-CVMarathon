# DL-CVMarathon

深度學習與電腦視覺百日馬拉松課程作業集，涵蓋從基礎機器學習到 CNN 的完整學習歷程。

## 課程內容

| 階段 | 天數 | 主題 |
|------|------|------|
| 基礎統計與特徵工程 | Day 001 - 050 | MSE/MAE、特徵處理、Blending、Stacking |
| 非監督式學習 | Day 054 - 063 | K-Means、階層式分群、PCA、t-SNE |
| 深度學習基礎 | Day 064 - 075 | Keras 入門、Sequential/Functional API、MLP |
| 深度學習進階 | Day 076 - 091 | Optimizer、Batch Normalization、正則化 |
| CNN 與遷移學習 | Day 093 - 105 | CNN 架構、Transfer Learning、ConvNetJS |

## 技術棧

- Python / Jupyter Notebook
- NumPy / Matplotlib
- Keras / TensorFlow
- scikit-learn

├── Day_064 ~ Day_075    # 深度學習基礎
